# 别名和称号导航

本目录保存别名、称号、旧 key 和阶段名到已知档案的跳转关系。

## 目录

- [characters](characters/README.md)：人物固定别名、称号和阶段导航 41 条；另有 [上下文称谓消歧指南](characters/contextual-title-guide.md)。

## 使用规则

- 命中固定别名文件后继续读取指向的人物档案。
- 别名文件只负责导航，不替代人物正文。
- `皇后`、`陛下`、`殿下`、`楼主`、`姐姐`、`主子` 等上下文称谓必须按当前视频、路线、说话人和前后节点消歧，不做固定映射；其中“说话人 + 称谓”可作为关系型线索。
